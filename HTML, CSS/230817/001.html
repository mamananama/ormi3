HTML 기본 요소(Element) 분석

요소의 분석

요소: <p class="web-tech">웹의 대표적인 기술 HTML/CSS/JS</p>

  • └ 여는 태그: <p class="web-tech">

  •   └ 속성명: class

  •   └ 속성 값: "web-tech"

  • └ 콘텐츠: 웹의 대표적인 기술 HTML/CSS/JS

  • └ 닫는 태그: </p>


요소의 포함

CODE <p>
 요소 안으로 요소</strong>를 넣을 수 있습니다.
</p>;
RESULT

요소 안으로 다른 요소를 넣을 수 있습니다.

잘못 사용된 예 <p>
 요소 안으로 요소를 넣을 수 있습니다.
</p>
 </strong>;


주석

VSCOD 사용시, 단축키 Ctrl+/로 주석 토글 가능.


HTML 계층 구조

CODE <html>
<head>
</head>
<body>
  <div>
  <p>
   <ul>
    <li> "Content1" </li>
   </ul>
  </p>
  <p> "Content2" </p>
 </div>
</body>
</html>
"Content1"이 속해 있는 요소를 기준,
부모는 <ul> 태그
조상은 <p> 태그

 └<body>
