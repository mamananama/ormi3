<head>

  • 메타 데이터를 담는 공간/HTML 문서 정보가 들어있다.
  • 검색엔진을 위한 다양한 정보 -- 메타 데이터를 통해 정보얻음
  • 웹 사이트 이용자들에게 필요한 title, favicon, viewport 등의 정보
  • <meta charset="">: 문자 인코딩 설정
  • <meta name="viewport" ... >: 뷰포트 제어 - 페이지 보이는 범위 결정
  • <link>: 현재 문서와 외부 리소스와의 관계 명시, 외부 스타일 시트/폰트/파비콘 연결

  • 사용자에게 보이는 영역
  • HTML 문서의 본문
